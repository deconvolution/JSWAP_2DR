## Installation
Install Julia on OS first. Then, in an IDE with Julia installed (e.g., [Atom](https://atom.io/)), one can install this package by
```julia
julia> ]
(@v1.6) pkg> add https://github.com/deconvolution/JSWAP_2DR
```
